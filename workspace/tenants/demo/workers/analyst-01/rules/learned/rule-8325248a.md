# When defining tools for LLM function calling, always explicitly populate the 'type' field (e.g., set it to 'function') and ensure it is not empty.

API providers like OpenAI (and wrappers like LiteLLM) enforce strict schema validation for tool definitions; omitting the 'type' or leaving it empty causes a BadRequestError, preventing the request from being processed.
